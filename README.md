# Sorting algorithms


## Quadratic sorting algorithms

- #### Bubble Sort
Asymptotic computational complexity: *O(n<sup>2</sup>)*

- #### Selection Sort
Asymptotic computational complexity: *O(n<sup>2</sup>)*

- #### Insertion Sort
Asymptotic computational complexity: *O(n<sup>2</sup>)*

- #### Comb Cort
Asymptotic computational complexity: *O(n<sup>2</sup>)*


## Efficient sorting algorithms

- #### Merge Sort
Asymptotic computational complexity: *O(n log n)*

- #### Counting Sort
Asymptotic computational complexity: *O(n + C)*
